<'!-- --'> Cria um comentário em qualquer parte do código-fonte.
<'html'><'/html'> Sempre no começo e no fim da página web. Ela limita onde o código começa e termina.
<'head'><'/head'> Definir o começo e o fim do cabeçalho de um documento html.
<'meta'> Dados basicos sobre o documento, como autor e data de crição
<'style'><'/style'> Define o estilo de parte ou de todo o conteúdo da página web.
<'noscript'><'/noscript'> Permite a adição de scripts à página web.
<'title'><'/title'> Define o título do documento HTML. Esse é o nome que é exibido no topo da janela do navegador quando a página é aberta
<'body'><'/body'> Indica para o navegador o corpo do documento HTML, ou seja, a parte principal da página web.
<'ol'><'/ol'> Cria listas ordenadas. Deve ser acompanhado de start, para indicar o começo da lista e type, para apontar o tipo de caractere que será ordenado.
<'menu'><'/menu'> Cria um menu com itens personalizados
<'dir'><'/dir'> Lista diretórios em um documento HTML.
<'code'><'/code'> Formata o texto como código.
<'samp'><'/samp'> Define um texto de amostra
<'cite'><'/cite'> Formata citações.
<'div'><'/div'> Define um bloco inteiro de conteudo. Muito util para organizar elementos na página
<'font'><'/font'> Define o tipo de fonte
<'img'><'/img'> Permite inserir imagens na página web.
<'frame'><'/frame'> Cria frames para a exibição de conteúdos.
<'frameset'><'/frameset'> Define um tipo de frameset.
<'table'><'/table'> Cria tabelas para a exibição de itens.
<'caption'><'/caption'> Permite adicionar legendas para algum elemento da página web.
'class=”…“' Atribui uma classe ao elemento (uma classe pode ser utilizada para um ou mais elementos);
<'form'><'/form'> Cria um formulario
<'label'><'/label> Define um rotulo para elementos do formulario
<'input'> Entrada de dados pode ter várias formas
<'input type="checkbox"> Caixa de seleção
<'input type="radio"> Elemento circular para seleção
<'input type="email"> Elemento de inserção de email
<'input type="password"> Elemento de inserção de senha
<'section'><'/section'> Cria uma seção no código
<'input type="submit"> Exibe um botão enviar (para enviar o formulário)
<'input type="button"> Exibe um botão clicável
<'table'> <'/table'> Cria uma tabela
<'tr'> <'/tr'> Cria uma linha da tabela
<'td'> <'/td'> Cria uma célula da tabela
<'th'> <'/th'> Transforma uma célula em título
